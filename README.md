# Snake Game

## Description
A simple Snake Game developed using Java Swing. The player controls the snake using arrow keys and collects food to increase the score.

## Features
- Keyboard controls
- Food generation
- Score tracking
- Real-time gameplay
- User-friendly interface

## Technologies Used
- Java
- Swing
- AWT

## How to Run
1. Download the project.
2. Open in Eclipse or IntelliJ IDEA.
3. Run `SnakeGame.java`.
